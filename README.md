# video_ai_admin

This template should help get you started developing with video_ai_admin.

## Project Setup

```sh
pnpm
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Type-Check, Compile and Minify for Production

```sh
pnpm build
```

### Lint with [ESLint](https://eslint.org/)

```sh
pnpm lint
```
